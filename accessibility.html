User Experience Model (v3.0)

Accessibility features

Every effort has been made to make the UXM accessible and easy to use for everyone, no matter what browser you choose to use, and whether or not you have any disabilities.

Accessibility standards

The layout of the UXM takes into account users who are blind or visually impaired. It is fully compatible with popular screen reading software.

In technical language, all pages on this site should be valid to Hyper Text Mark-up Language (XHTML) 1.0 Transitional and use Cascading Style Sheets (CSS).

This website conforms to the UK government guidelines for websites.

It also follows the Worldwide Web Consortium's (W3C) Web Content Accessibility Guidelines 1.0 and strives to meet level double-A checkpoints.

Maintaining an accessible site is an ongoing process and nidirect is continually working to offer a user friendly experience. However, if you have any problems using this website please contact nidirect Transactional Services Manager.
