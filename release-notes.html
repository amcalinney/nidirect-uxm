Release notes

Release notes for each version of the UXM are listed below.

ETA: Autumn 2019 (v3.0)

  • Complete redesign using Bootstrap 4
  • Example text
  • Example text
  • Example text
  • Example text

13 May 2019 (v2.2)

  • Added page for Usability Testing
  • Removed 'Back to Top' feature
  • Download the Developer's Toolkit (v2.2)

30 October 2017 (v2.1)

  • Miscellaneous updates based on the recommendations of an Accessibility Audit to meet WGAC AA standards, including:
    • minor changes to colours to meet contrast requirements
    • updating the styling of external links
    • an accessibility statement
  • Design patterns have been categorised into 2 areas
    • Control patterns
    • Information patterns
  • Added information patterns on the use of description lists for
    • Summary information
    • Application summaries
  • Added a design pattern for a search control
  • Updated 'Writing guide' to include a design for confirmation pages
  • Updated the calendar control to restyle the button
  • Updated javascript and css references to improve performance
  • Download the Developer's Toolkit (v2.1)

14 November 2016 (v2.0)

  • Complete re-design using Bootstrap (v3.3.7)
  • Aligned to nidirect theme
  • Controls have been categorised into 3 areas
    • Input controls
    • Navigation controls
    • Information controls
  • Progress trackers are no longer compulsory
  • Information relating to the design and layout has been moved to a new page
  • The link to the Resources page has been relocated from the menu to the footer
  • Release notes can now be found within the Resources page
  • Download the Developer's Toolkit (v2.0)

27 May 2016 (v1.4)

  • Updated principles
  • Updated form elements section with a redesign of radio button and checkbox controls
  • Updated validation and error messages section with a redesign of validation display
  • Amended applying the UXM section to update browser support
  • Download Developer's Toolkit (v1.4)

8 September 2015 (v1.3)

  • Added a new error pages section - which includes examples of how errors should be presented to users.
  • Added a new pagination section - which shows how to display pagination within an application.
  • Download Developer's Toolkit (v1.3)

15 December 2014 (v1.2)

  • Added a new date controls section - which includes guidance on memorable dates, approximate dates, calendar controls and formatting dates.
  • In the forms section:
    • updated the text area example to include a character count; and
    • added two examples of progressive disclosure - using radio buttons and a drop down.
  • In the help content section - added an example of progressive help.
  • In the policies section - added text to reinforce the mandatory use of the Information Assurance Framework
  • Download Developer's Toolkit (v1.2)

24 January 2014 (v1.1)

  • Added a note to the responsive layout section to advise that the live grid system is not compatible with this version of the UXM. If you wish to implement a grid system please use the fluid grid.
  • Reduced the line height of <textarea> to 20px
  • Reduced the spacing between <h1> and <h2> in the grid layout
  • Fixed some issues with button sizes
  • Changed css to form layout if there is no <fieldset> markup
  • Added validation examples for textareas, radio buttons and selects
  • Changed the class name for 'checkbox-error-inner' to 'error-inner' to improve semantics for radio button validation
  • Changed 'e.g.' to 'eg' throughout the UXM to match the nidirect writing style
  • Added guidance on inline validation for the log in design pattern
  • Download Developer's Toolkit (v1.1)

2 December 2013 (v1.0)

  • Added guidance, code and screenshots for responsive layout
  • Added specific guidance on responsive layout requirements for the following components and design patterns:
    • Top navigation and branding
    • Footer
    • Headings
    • Form elements
    • Validation and error messages
  • Added the 'nidirect-responsive.css' stylesheet
  • Numerous changes to 'nidirect.css' related to the responsive layout
  • Added drop down menu option to the top navigation
  • Moved the guidance for buttons out of a table to improve readability on smaller screens
  • Added additional class names for the call to action and warning buttons to improve semantics
  • Added the information notice component
  • Fixed some character encoding issues with the display of apostrophes on the principles section
  • Added additional guidance around the use of robots instruction to the head section
  • Added favicon reference for iOS devices to the head section
  • Removed reference to the 'content to follow' design patterns. Design patterns for search interfaces and summary / confirmation page will be included in a subsequent version of the UXM.
  • Download Developer's Toolkit (v1.0)

28 October 2013 (beta v0.9.2)

  • Added guidance, code and examples for progress trackers
  • Added guidance for skip navigation
  • Updated the guidance for primary heading to include skip navigation
  • Renamed the metadata section to head section
  • Replaced the branding header, top nav and footer divs with appropriate html5 elements (i.e. <header>, <nav> and <footer>)
  • Added a html5 shiv with associated guidance to the head section
  • Added the nidirect Identity Guidelines document to the >policies section
  • Added a screen reader only class (.sr-only) to nidirect.css
  • Added outline to the nidirect logo for keyboard focus (.branding a:focus) to nidirect.css

15 October 2013 (beta v0.9.1)

  • Added (UXM) to the page/window title on all pages
  • Removed the version number from the link text of policy documents (the version number is on the document cover page and the filename). New versions added to the UXM will be listed in the release notes.
  • Added additional guidance for drop down and radio button form elements
  • Updated the alt text of the nidirect logo
  • Removed the 'robots - noindex, nofollow' metadata in this UXM. Please note that the requirement to add the robots metadata to transactional services still exists

8 October 2013 (beta v0.9)

  • Beta version launched
